# GBA_Project

A Game Boy Advance project built with devkitPro and Butano.

## Environment

- devkitPro: `C:/devkitPro`
- Project: `G:/GBA_Project`

## Compile

Run this command in the project directory:

```powershell
make -j8
```

## Run

After compiling, run the generated ROM:

```powershell
.\GBA_Project.gba
```
